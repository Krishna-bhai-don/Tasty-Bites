# 🍽️ Tasty Bites Restaurant Website

## 📌 Project Overview

**Tasty Bites Restaurant Website** is a simple and responsive restaurant landing page built using **HTML, CSS, and JavaScript**. The website showcases a restaurant's menu, information about the restaurant, and a contact form for customer inquiries.

This project is suitable for beginners who want to learn front-end web development and website design.

---

## ✨ Features

* Responsive navigation bar
* Attractive hero section with background image
* Restaurant menu display
* About Us section
* Contact form with validation
* Interactive JavaScript alerts
* Clean and modern user interface

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the website
* **CSS3** – Styling and layout
* **JavaScript** – Interactive functionality

---

## 📂 Website Sections

### 1. Header & Navigation

* Restaurant logo/name
* Navigation links:

  * Home
  * Menu
  * About
  * Contact

### 2. Hero Section

* Welcome message
* Restaurant tagline
* "Book a Table" button

### 3. Menu Section

Displays food items and prices:

| Item        | Price |
| ----------- | ----- |
| Pizza       | $10   |
| Burger      | $8    |
| Pasta       | $12   |
| Cold Coffee | $5    |

### 4. About Section

Provides information about the restaurant and its services.

### 5. Contact Section

Includes:

* Name field
* Email field
* Message field
* Submit button

### 6. Footer

Contains copyright information.

---

## ⚙️ JavaScript Functionality

### Book a Table Button

When clicked, displays:

```javascript
alert("Thank you! Table booking service coming soon.");
```

### Contact Form Submission

* Prevents page refresh
* Displays success message
* Resets form fields

```javascript
alert("Message Sent Successfully!");
```

---

## 🚀 How to Run the Project

1. Copy the code into a file named `index.html`.
2. Save the file.
3. Open it in any modern web browser:

   * Google Chrome
   * Microsoft Edge
   * Mozilla Firefox
   * Safari

No additional installation or dependencies are required.
